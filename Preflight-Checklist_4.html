    One-time setup

    Connect a Google Form

    Completed checklists can be sent straight into a Google Form (and its response spreadsheet). Set it up once and every operator using this page can submit records.

    1. Create a Google Form. Add one Short answer question for each field below — use the exact code as the question title (the last one, RECORD, should be a Paragraph question).
    2. In the form editor, open the ⋮ menu → "Get pre-filled link."
    3. In the pre-filled view, type each question's code into its own answer box (type PRODUCT into the Product question, and so on), then click Get link → Copy link.
    4. Paste that link here:
    PRODUCT MODEL VARIANT TEAM STARTBY STARTDATE STARTTIME ENDTIME ASMDATE COMPLETEDBY COMPDATE LINE INITIALS FAI LAI PROGRESS RECORD
    You can skip fields you don't need — only the codes you fill in will be mapped.
